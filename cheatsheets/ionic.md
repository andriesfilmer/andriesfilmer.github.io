# Ionic cheatsheet

## First steps

    sudo npm install -g fsevents
    sudo npm install -g @ionic/app-scripts


To run or build your app for production, run

    ionic cordova run android --prod --release
    # or
    ionic cordova build android --prod --release


To run in browser

    ionic serve
